# ITS Africa Conference 2027 — Brand Guidelines

The brand and design system for the **ITS Africa Conference 2027** (Cape Town, 9–10 March 2027).

**Live site:** https://its-congresses.github.io/itsac-brand-guidelines/

This repository hosts the guidelines as a self-contained static site: the single source of truth for logos, colour, typography, the Ndebele-inspired pattern system, templates and downloadable assets. It's the delivery vehicle for the ITSAC and ERTICO/Backbone marketing teams.

## What's inside

- `index.html` — the guidelines site (logo usage, click-to-copy colour palette, typography, full pattern system, partner lockup, email signature example, downloads).
- `assets/logos/` — logo set: three layouts (landscape, portrait, square) across mono, white-knockout, boxed and transparent treatments.
- `assets/pattern/` — the modular pattern system: five modules, 24 approved colourways, master strips and proofs.
- `assets/banners/` — banner assets.
- `downloads/` — the Word letter template (`.docx` / `.dotx`) and the email signature example + strip.

## Working principle

**Zero creative decisions for the team.** End users edit text and swap photos — nothing else. Every colour, layout and lockup is already decided and locked into the templates and this site.

## Deployment

Served via GitHub Pages from the `main` branch (root). Push to `main` and the live site updates automatically.
